# Suisse-17 - Niches

## Objetivo

Este documento define las áreas de exploración iniciales de Suisse-17.

Los nichos seleccionados deben cumplir al menos una de las siguientes condiciones:

- experiencia profesional real
- experiencia personal real
- capacidad de generar contenido diferencial
- potencial SEO
- potencial de monetización
- sostenibilidad a largo plazo

---

# Reglas de expansión

Un nuevo nicho solo podrá incorporarse cuando:

- exista una ventaja competitiva clara
- exista potencial de monetización
- exista tiempo disponible para mantenerlo
- no comprometa los nichos existentes

La calidad tendrá prioridad sobre el número de proyectos.
