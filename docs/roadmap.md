- principles.md

# Criterios de éxito

SEO:

- tráfico orgánico creciente
- posicionamiento estable

Audiencia:

- newsletter
- comunidad

Monetización:

- ingresos diversificados

Operación:

- mantenimiento reducido
- máxima reutilización

---

# Criterio de abandono

Un proyecto podrá pausarse si:

- no muestra señales de crecimiento
- consume demasiado tiempo
- carece de ventaja competitiva
- existen mejores oportunidades

El objetivo es optimizar activos, no acumular proyectos.
